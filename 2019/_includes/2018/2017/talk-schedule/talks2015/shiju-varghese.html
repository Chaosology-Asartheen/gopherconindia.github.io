Building RESTful Services With Go and MongoDB

Friday 20th February, 2015    1:50 pm to 2:15 pm (IST)

This session will demonstrate, how to build REST based web services in Go with MongoDB as a persistence storage. This session will also leverage Go package mgo for working with MongoDB from Go apps.

Audience : Intermediate to Advanced developers with experience in any of the programming languages - Go, Java, C#, Node.js, Ruby and Python.

This session lets the web developers to write RESTful services in Go and can learn how to work with a popular NoSQL database MongoDB by using Go package mgo.
