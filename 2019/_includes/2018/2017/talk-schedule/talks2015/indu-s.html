Go for everything!

Friday 20th February, 2015    10:45 am to 11:10 am (IST)

This talk will cover what Go as a language with its set of libraries has to offer for various domains from machine learning to bioinformatics, from API servers to the Internet of Things. A survey of interesting open source projects where the Go community can contribute.

Indu S. @induhub
Indu is a developer who has delved in to Go programming recently and explores Go for various software solutions she needs to write for various domains. She started her career as a Python developer and has experience in writing web applications and building scalable cluster architectures from elastic search to message queues. She has some devops experience using Docker. She plans to be a Go evangelist and an active member of the Go community!
