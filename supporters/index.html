Sourcegraph is a code search engine that shows you documentation and real-world usage examples for hundreds of thousands of libraries written in Python, JavaScript, Ruby, and Go. Unlike other code search engines and Google, Sourcegraph analyzes source code. It automatically identifies functions, classes, etc., in code and finds all references to them (even from other repositories), which means you can quickly find accurate and comprehensive docs and examples.
