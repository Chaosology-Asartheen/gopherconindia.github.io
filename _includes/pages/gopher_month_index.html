GopherMonth

Go, also called golang, is an open source programming language (though initially developed at Google) that makes it easy to build simple, reliable, and efficient software.

A Gopher stands for the Go project and Go programmers everywhere.

Go has:

  • Simple, minimal syntax
  • Garbage collection built-in
  • A flexible interface system
  • Easy concurrency support via goroutines
  • Fast compilation times
  • Simple compile build/run procedures
  • Statically linked binaries that are simple to deploy
  • Fun to write
  • Much much more..

GopherMonth is a month-long series of online hackathons based on the Go programming language. It is aimed at encouraging new developers come aboard and try Go for the first time, and for seasoned gophers to show off their skills and earn some glory. As part of GopherMonth, three hackathons will be hosted on the DoSelect platform, starting from January 15th, 2016. More details can be found here.
