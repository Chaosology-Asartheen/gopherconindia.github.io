Develop and manage large scale of Go web application

Friday 20th February, 2015    12:00 pm to 12:30 pm (IST)

Gogs, as a self-hosted Git service web application, it is very different from those applications that only work on command line interface or as services. Gogs is large scale web application whose codebase is rapidly growing, there are many lessons from developing and managing this kind of application. Gogs also faces challenges of distributing different versions to customers, which requires a good dependency management mechanism.
